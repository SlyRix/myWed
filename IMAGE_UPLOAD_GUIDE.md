## ğŸ› Troubleshooting

### "Image not loading"
- **Cause:** File not yet uploaded to server
- **Fix:** Send the image file to your developer

### "Changes not appearing"
- **Cause:** Browser cache
- **Fix:** Hard refresh: `Ctrl+Shift+R` (Windows) or `Cmd+Shift+R` (Mac)

### "Image looks stretched"
- **Cause:** Wrong image dimensions
- **Fix:** Use landscape photos (wider than tall) for hero images

### "Upload button not working"
- **Cause:** File too large or wrong format
- **Fix:** Compress the image or use JPG format
